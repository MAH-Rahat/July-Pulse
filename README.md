# July Pulse üáßüá© 
**Capturing the Rhythm of the Revolution.**

July Pulse is a news portal developed as part of a **Prompt Engineering Competition**. The project was built to demonstrate the ability to rapidly prototype a theme-based web application, focusing on the historic **July 2024 Student-People Uprising** in Bangladesh.

---

## üèÜ Competition Context
This project was developed under competition constraints where the challenge was to utilize **Prompt Engineering** and rapid development workflows to create a meaningful digital product. 
- **Theme:** Bangladesh July-2024 Situation.
- **Project Goal:** To create a functional, high-impact news archive that documents the uprising through a modern "Gen Z" lens.
- **Core Focus:** Clean UI/UX, responsive navigation, and a robust content management structure.

---

## üöÄ Overview
During the events of July 2024, information was the most powerful tool. **July Pulse** serves as a digital archive, featuring a dark-themed, high-contrast interface to reflect the gravity of the situation while maintaining modern web standards for performance.

### Key Features
- **Dynamic News Feed:** Categorized sections for Politics, Sports, and Entertainment.
- **"July Martyrs" Archive:** A dedicated space recognizing the "July Warriors" of the movement.
- **Admin Functionality:** Integrated PHP scripts (`add_news.php`, `submit_article.php`) for content management.
- **Responsive Navigation:** Optimized for mobile-first users.
- **Clean UI:** High-visibility action buttons and a structured grid layout for news cards.

---

## üõ†Ô∏è Tech Stack
This project was built using a mix of front-end and back-end technologies:

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)

## üìÅ Project Structure
- `/assets`: Media files and images of the uprising.
- `/includes`: Reusable components (headers, footers, common functions).
- `index.html`: Main landing page showcasing featured news cards.
- `view_article.html`: Dynamic template for reading full stories.
- `add_news.js`: Frontend validation and logic for news entry.
- `config.php`: Database connection and environment settings.
